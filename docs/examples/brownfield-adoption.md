# Brownfield Adoption Example

User asks Codex to adopt an existing game project.

Expected flow:

1. Load `game-studio-onboarding`.
2. Inspect docs, build files, project structure, and playable evidence.
3. Identify the phase and missing artifacts.
4. Route to `game-studio-flow`, `game-studio-architecture`, or `game-studio-qa`.
5. Produce one small next action with a validation path.
