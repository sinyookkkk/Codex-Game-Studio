# UX Review Example

User asks for a UX review of menus or HUD.

Expected flow:

1. Load `game-studio-design` or `game-studio-gate` depending on whether a verdict is requested.
2. Apply UX designer, UI programmer, accessibility specialist, and QA lead lenses.
3. Check first-time comprehension, feedback, controls, layout, and accessibility.
4. Return fixes ordered by player impact and validation cost.
